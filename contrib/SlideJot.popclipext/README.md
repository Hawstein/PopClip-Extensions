# SlideJot

Send selected text to SlideJot.

## Features

- **Click**: Create a new Jot with selected text
- **Option (⌥) + Click**: Append selected text to end of current active Jot
- **Shift (⇧) + Click**: Prepend selected text to beginning of current active Jot

## Requirements

- [SlideJot](https://slidejot.com) app installed
